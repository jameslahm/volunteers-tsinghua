## A mini-program for volunteer activity management

- Run:
  - PC: 
    ```bash
    cd webapp && make runserver
    ```
  - WX:
    ```bash
    cd wxapp && npm run build
    ```

- Test:
  ```bash
  cd webapp && make test
  ```

- Init database:
  ```bash
  cd webapp && make init
  ```

- Migrate database:
  ```bash
  cd webapp && make migrate
  ```

- Upgrade:
  ```bash
  cd webapp && make upgrade
  ```
